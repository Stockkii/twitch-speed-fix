Automaticly resets the playback-speed of the Twitch-Video-Player to prevent audio bugs in low-latency-mode.

How to install:

- Create the folder "twitch-speed-fix" somewhere you can remember and download the given files into that folder
- Open your desired chromium-based browser and go to "*insertbrowsername*://extensions" (as example chrome://extensions or brave://extensions)
- Enable the developer mode (usually in the top right corner)
- Click on "Load unpacked" (usually in the top left corner)
- Navigate to the location of the extension folder and select it
